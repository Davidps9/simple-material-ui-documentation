## Calling the component

```tsx
<SCard  title="This is the title" subtitle="This is the subtitle" image="image-path" />
```
<SCard title="This is the title" subtitle="This is the subtitle" image="/_astro/blue-drag.CEHiWItG_ZWPpBR.webp" />

## Props

### Size
> The size of the card, the component expects a string and if it is not passed, its default value is ```medium```

    <TabItem label="Medium">

        <SCard title="This is the title" subtitle="This is the subtitle"  size="medium" text="This is the text"  />
    </TabItem>
    <TabItem label="Large">
        ```tsx
        <SCard title="This is the title" subtitle="This is the subtitle"  size="large" text="This is the text" />
        ```

        <SCard title="This is the title" subtitle="This is the subtitle"  size="large" text="This is the text" />
    </TabItem>
</Tabs>

### Variant
>The type of card, the component expects one of three values, elevated, filled or outlined and if not passed, its value is set to `filled`

<Tabs>
    <TabItem label="elevated">


    </TabItem>
    <TabItem label="filled">
        ```tsx
        <SCard title="This is the title" subtitle="This is the subtitle"  variant="filled"  text="This is the text" />
        ```
        <SCard title="This is the title" subtitle="This is the subtitle"  variant="filled"  text="This is the text"  />
    </TabItem>
    <TabItem label="outlined">
        ```tsx
        <SCard title="This is the title" variant="outlined" subtitle="This is the subtitle"  borderColor="gray" text="This is the text" />
        ```

        <SCard title="This is the title" variant="outlined" subtitle="This is the subtitle"  borderColor="gray" text="This is the text" />
    </TabItem>
</Tabs>

### Title
> The title of the card, the component expects a string and if it is not passed it will not be rendered

```tsx
<SCard  title="This is the title" />
```
<SCard title="This is the title" />

### Subtitle
> The subtitle of the card, the component expects a string and if it is not passed it will not be rendered

```tsx
<SCard  subtitle="This is the subtitle" />
```
<SCard subtitle="This is the subtitle" />

### Image
> The image of the card, the component expects a string, representing the path to the image, and if it is not passed it will not be rendered

```tsx
<SCard  image="image-path" />
```
<SCard image="/_astro/blue-drag.CEHiWItG_ZWPpBR.webp" />

### Text
> The text of the card, the component expects a string and if it is not passed it will not be rendered

<SCard title="This is the title" subtitle="This is the subtitle"  text="This is the text" />

### Limit text
> The limit text of the card, the component expects a number and if it is not passed, its default value is unlimited. This prop adds ```...``` at the end of the text if its passed a number greater than 0.

<Tabs>
    <TabItem label="Unlimited">

        <SCard text="This text is unlimited" limitText={0} />

        ```tsx

        ```
